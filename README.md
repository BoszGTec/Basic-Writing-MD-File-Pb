# Basic-Writing-MD-File-Pb
การเขียนไฟล์md
> ## การเขียน header :
  + ใช้รูปแบบการเขียนดังนี้ ```# ข้อความ```
  + ตัวอย่าง 
    # ใช้ ```#``` 1ตัว
    ## ใช้ ```#``` 2ตัว
    ### ใช้ ```#``` 3ตัว
    #### ใช้ ```#``` 4ตัว
    ##### ใช้ ```#``` 5ตัว
    ###### ใช้ ```#``` 6ตัว (มากที่สุดแล้ว)
> การเขียนตัวหนา/ตัวเอียง
  + การเขียนตัวหนาใช้รูปแบบการเขียนดังนี้ : 
    ```**ข้อความ**``` หรือ ```<b>ข้อความ</b>```
  + ตัวอย่าง  
    **ข้อความ**
